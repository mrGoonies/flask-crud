# Proyecto Flask

En este proyecto hare uso del micro-framework Flask para crear APIs con el objetivo de aprender su funcionamiento y características principales, adicionalmente se implementaran buenas prácticas en el desarrollo de APIs RESTful. Adicionalmente, se utilizará Docker para la contenerización de la aplicación, facilitando su despliegue y gestión en diferentes entornos.

## Contexto App

Una empresa posee varias tiendas junto a información de productos, precios y stock. Se requiere desarrollar una API RESTful que permita gestionar esta información de manera eficiente. La API debe permitir realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre las tiendas y los productos, así como consultar el stock disponible en cada tienda por producto.

## Objetivo de la API

El objetivo es poder poner en practica los conceptos fundamentales de Flask y el desarrollo de APIs RESTful, para entregar una app simple pero funcional para el taller de testing en Flask que esta pensado en ser un aporte para el curso de verano enfocado en el desarrollo Full-Stack con Python impartido por [Boolean Academia](https://boolean.cl/cursos/python-fullstack/).


### Herramientas utilizadas

- **Flask**: Micro-framework para el desarrollo de aplicaciones web en Python.
- **Docker**: Plataforma para la creación, despliegue y gestión de contenedores.
- **SQLAlchemy**: ORM para la gestión de bases de datos en Python.
- **uv**: Gestor de proyectos y dependencias en Python.
